# enjoy-project-tool

- [x] 创建项目模板，目前仅支持vue3,node-command
- [ ] 自定义eslint,husky等安装选项

## 命令

| 命令       | 含义                                                    |
| ---------- | ------------------------------------------------------- |
| create        | 创建模板         |

## 使用

```shell
npm i enjoy-project-tool -g
```

```shell
enjoy create
```
