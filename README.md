# olkj-cli

- [x] 创建项目模板，目前支持 vue3,node-command,rollup-package
- [x] 自定义选择模板版本
- [x] 初始化 GIT
- [x] 自定义 UI 框架
- [x] React 支持
- [x] 支持给项目添加 workflow ci

## 特性

- 项目模板下载&安装国内提速
- 开箱即用的 github ci 配置

## 命令

| 命令     | 含义             |
| -------- | ---------------- |
| create   | 创建模板         |
| workflow | 创建 ci 配置文件 |

## 使用

```shell
npm i olkj-cli -g
```

```shell
olkj create
```
