# enjoy-project-tool

- [x] 创建项目模板，目前支持vue3,node-command,rollup-package
- [x] 自定义选择模板版本
- [x] 初始化GIT
- [x] 自定义UI框架
- [ ] React支持
## 特性

- vue模板下载&安装国内提速（淘宝源⚡️）
## 命令

| 命令       | 含义                                                    |
| ---------- | ------------------------------------------------------- |
| create        | 创建模板         |
| workflow | ci配置文件（正在开发中）

## 使用

```shell
npm i enjoy-project-tool -g
```

```shell
enjoy create
```
